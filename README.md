# XSS-Platform
### How to use?
```
# git clone https://github.com/SilNex/XSS-Platform  
# docker-compose up
```

### Access
> http://localhost:8080/  

### Query
#### Recommend
> http://localhost:8080/  
> POST data( XSS=g0x55&url={XSS_Link}&flag={flag} )

or

> http://localhost:8080/?XSS=g0x55&url={XSS_Link}&flag={flag}
